# Asset Tasks Among Us - README

## Overview
Welcome to **Asset Tasks Among Us**! This project is a custom asset-based setup that includes UI elements and additional gameplay mechanics inspired by **Among Us**. The project allows users to interact with assets dynamically.

## Features
- **Custom UI & Design**: A modern interface for easy navigation.
- **Game Enhancements**: Custom mechanics to enhance gameplay.
- **Dynamic Asset Interaction**: Interact with game assets in real-time.

## Installation
### Prerequisites
- **Python 3.8+** must be installed on your system.
- Ensure `pip` is updated to the latest version:
  ```bash
  python -m pip install --upgrade pip
  ```

### Steps to Run:
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/MrOwner-alt/Tasksamongus.git
   cd Asset-Tasksamongus
   ```
2. **Install Dependencies:**
   Run the following command to install all required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. **Follow the On-Screen Instructions** to configure your profile and settings.

### Running the Game
After setup, you can start the game by running:
```bash
python main.pyw
```

## Troubleshooting
- **Dependency Issues**: Ensure all dependencies are installed properly using `pip install -r requirements.txt`.
- **Permission Errors**: Run the script in administrator mode if you encounter permission errors.
- **Python Version**: Verify that you are using Python 3.8 or higher.
